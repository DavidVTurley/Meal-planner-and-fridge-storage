# Implementation History

This folder stores append-only records of implementation plans for this repository.

## File Naming Convention

- `YYYY-MM-DD_HHMM_<short-kebab-title>.md`
- Timezone: Europe/Amsterdam.

## Required Sections Per Plan File

- `Title`
- `Date/time`
- `Context/request`
- Full `<proposed_plan>` content

## Policy

- Never overwrite existing plan files.
- Add a new timestamped file for each new implementation plan.
